# Dataset Collection
Source corpus and goldset are collected from https://github.com/masud-technope/BLIZZARD-Replication-Package-ESEC-FSE2018/tree/master (Improving IR-Based Bug Localization with Context-Aware Query Reformulation)

AuthorHistory.zip contains the authors' information for all six subject systems

StackTrace.zip contains the stack trace information for all six subject systems

VersionHistory.zip contains version history information for all six subject systems

# Results produced by six information sources

TS -  Go to TS.zip. The result file for a given system will be as follows: system.txt (e.g., ecf.txt)

TC -  Go to TC-BFH.zip. The result file for a given system will be as follows: system_output_0.0_no_of_bug.txt (e.g., ecf_output_0.0_553.txt)

BFH -  Go to TC-BFH.zip. The result file for a given system will be as follows: system_output_1.0_no_of_bug.txt (e.g., ecf_output_1.0_553.txt)
